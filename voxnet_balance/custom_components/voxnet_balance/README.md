# Voxnet Balance (Home Assistant HACS Integration)

Интеграция для провайдера Voxnet / Lantek.

## Возможности
- Баланс счёта
- Лицевой счёт
- Статус интернета
- Кредит
- Заблокированные средства

## Установка (HACS)
1. Добавить репозиторий в HACS (Custom repository)
2. Выбрать "Integration"
3. Установить Voxnet Balance
4. Перезагрузить Home Assistant
5. Добавить через UI
